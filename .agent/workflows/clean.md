# Workflow: Clean

Nettoie le code pour un projet propre et maintenable.

## Étapes

### 1. Supprimer les console.log de debug
Rechercher et supprimer les `console.log` de debug (garder les erreurs et warnings importants).

### 2. Vérifier les imports inutilisés
Analyser les fichiers JS et retirer les imports non utilisés.

### 3. Nettoyer le CSS mort
Identifier les classes CSS qui ne sont plus utilisées dans le HTML/JS.

### 4. Supprimer le code commenté obsolète
Retirer les gros blocs de code commentés qui ne sont plus pertinents.

### 5. Vérifier le formatage
S'assurer que le code est correctement indenté et formaté.

### 6. Rapport final
Lister les fichiers nettoyés et les modifications effectuées.
