              Guide de l'application

              Pour fonctionner, l'application a besoin d'un "cerveau" IA. Nous recommandons Google Gemini pour débuter car il est performant et gratuit.

              1
              Récupérer votre clé (Gratuit)

              Allez sur Google AI Studio et connectez-vous avec votre compte Google.

              Ouvrir Google AI Studio
              2
              Copier / Coller

              Cliquez sur "Create API key", copiez la clé (commence par AIza...), et collez-la dans les Paramètres de cette application.

              Alternative : Accéder à plus de modèles via OpenRouter

              OpenRouter est une passerelle unifiée qui vous donne accès à des dizaines de modèles IA (Mistral, DeepSeek, Claude, GPT-4, etc.) avec une seule clé API.

              • ✓ Crédit gratuit offert à l'inscription
              • ✓ Accès aux modèles les plus performants
              • ✓ Tarification à l'usage (très économique)
              Obtenir une clé OpenRouter
              Relais automatique : Si le modèle échoue (limite, surcharge...), l'app peut basculer vers un autre. Activable dans les Paramètres avancés.

              Vous avez deux options :

              • Mode Unitaire : Saisissez les notes d'un élève à la main pour tester.
              • Import en Masse (Recommandé) : Copiez-collez directement depuis Excel, Pronote ou un fichier CSV.

              L'application détecte automatiquement les colonnes. Si besoin, vous pourrez ajuster le "mappage" (quelle colonne correspond à quoi) avant de valider.

              L'IA s'adapte à votre style d'écriture via les Paramètres :

              • Ton et Longueur : Strict ou encourageant ? Concis ou détaillé ?

              Utilisez le bouton Aperçu dans les paramètres pour tester vos réglages en temps réel sans modifier vos vrais bulletins.
