Comment ça marche ?

Une interface simple pensée pour les enseignants, pas pour les informaticiens.

1

Importez vos notes

Copiez-collez vos listes depuis Pronote, Excel ou Sheets. L'outil détecte automatiquement les colonnes.

2

Paramétrez l'IA

Choisissez le ton (encouragement, fermeté), la longueur souhaitée et le modèle d'IA (Gemini, GPT-4, etc).

3

Exportez

Générez, ajustez si besoin et exportez vos appréciations prêtes à être collées dans votre logiciel scolaire.

Tout ce dont vous avez besoin

Puissant, flexible et respectueux de votre vie privée.

Multi-Modèles IA

Utilisez gratuitement Google Gemini intégré, ou connectez vos propres clés API (OpenAI, Anthropic). Compatible avec Ollama pour une utilisation 100% locale et hors-ligne.

Personnalisation

Définissez des règles précises : vouvoiement, tutoiement, structure des phrases, et vocabulaire spécifique.

100% Privé

Vos données élèves restent sur votre ordinateur. Aucune donnée n'est envoyée sur nos serveurs.

Formats Universels

Compatible CSV, Excel, et copier-coller direct depuis les principaux logiciels scolaires.

Historique & Journal de bord

Retrouvez facilement les appréciations générées. Tenez un journal de bord durant le trimestre pour nourrir l'IA avec des faits précis lors de la rédaction finale.

Questions Fréquentes

Tout ce que vous devez savoir avant de commencer.

Oui, Bulletin AI est un projet Open Source. Vous pouvez l'utiliser gratuitement avec Google Gemini (clé API gratuite) ou Ollama (local). Si vous utilisez OpenAI (GPT-4), vous payez uniquement votre consommation API directement à OpenAI.
Absolument. Bulletin AI est une application "client-side". Cela signifie que toutes les données (noms des élèves, notes) restent stockées localement dans votre navigateur. Rien n'est envoyé sur un serveur central chez nous. Seul le texte à générer est envoyé à l'IA de votre choix lors de la génération.
Bien sûr ! L'IA propose une base de rédaction, mais vous gardez la main totale. Vous pouvez éditer, reformuler ou compléter chaque appréciation avant de l'exporter.
C'est une application web, il n'y a rien à installer ! Cliquez simplement sur "Lancer l'App" et vous pouvez commencer immédiatement dans votre navigateur.
