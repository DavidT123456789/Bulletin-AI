Spécifications

Détails techniques et fonctionnement.

Les données sensibles (identités des élèves, notes brutes) sont stockées exclusivement dans le LocalStorage du navigateur. L'application applique une anonymisation stricte à la volée avant tout envoi vers les API d'inférence.
Compatible nativement avec Mistral AI (gratuit), Google Gemini et OpenRouter. Intégration complète avec Ollama pour une exécution 100% hors ligne (Air-Gapped).
L'application utilise des System Prompts calibrés pour garantir un ton professionnel. Les commandes (Concis, Détaillé, Encourageant) modifient dynamiquement les paramètres de génération.
Algorithme de détection automatique de colonnes compatible avec tout tableur (Excel, CSV, Pronote). Export en format texte brut formaté pour les logiciels de vie scolaire.
